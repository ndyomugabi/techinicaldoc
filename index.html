introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser)

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by

  • client side javascript extends the core language by supplying objects to control a browser and its document object model
  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server

    JavaScript and Java are similar in some ways but fundamentally different in some others.

    In contrast to Java's compile-time system of classes built by declarations, JavaScript supports a runtime system

    JavaScript is a very free-form language compared to Java. You do not have to declare all variables, classes, and methods.
